# Reference Tone Tuner (NVDA Add-on)

O **Reference Tone Tuner** é um add-on para o **NVDA** que reproduz **tons de referência** para você afinar o violão **de ouvido**.

> ⚠️ Este add-on **não detecta automaticamente** se a corda está acima ou abaixo da afinação correta.  
> A ideia é você ouvir o tom, comparar e ajustar manualmente.

---

## 🎵 Como usar

Abra em:  
**NVDA > Ferramentas > Reference Tone Tuner**  
ou use o atalho: **NVDA+Shift+V**

Na janela, utilize os seguintes comandos:

- **1 a 6** → toca uma corda (1 = Mi agudo … 6 = Mi grave)  
- **T** → toca todas as cordas (da 6ª para a 1ª)  
- **S** → acorde de Sol maior  
- **M** → acorde de Mi maior  
- **R** → repetir em loop (liga/desliga)  
- **F1** → ajuda com todos os atalhos  
- **Esc** → para o som (ou fecha a janela)
